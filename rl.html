3. Header and Layout Standards

All report headers must follow the structure below. Row 1 should contain the report title in 14pt bold. Row 2 contains the date range. Row 3 is blank (spacer). Data headers begin in Row 4.

Column A should be reserved for line item labels. Columns B onward contain data periods (quarters or months, left to right = oldest to newest).

4. Conditional Formatting Rules

Apply these conditional formatting rules to all financial data ranges:

ConditionFormat

YoY growth > 20%Green highlight

5. Sheet Structure

Standard workbooks should include the following sheets in order:

Summary — Executive summary with key metrics. P&L — Income statement. Balance Sheet — Assets, liabilities, equity. Cash Flow — Operating, investing, financing activities. Assumptions — All input assumptions in blue font. Appendix — Supporting data and calculations.

Note: These standards align with the Goldman Sachs and Morgan Stanley formatting conventions commonly used in investment banking. Deviations should be documented in the Appendix sheet.
